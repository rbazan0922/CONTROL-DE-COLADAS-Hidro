## Estructura
- `pages/` - Páginas HTML
- `JS/` - Scripts JavaScript
- `css/` - Estilos CSS
- `img/` - Imágenes

## Iniciar
1. Abrir `index.html` en navegador (o usar Live Server)
2. Ingresar usuario y contraseña
3. ¡Usar el sistema!

## Formularios Disponibles
- **Control Coladas H. Basculante** - Registro de coladas con basculante
- **Control Coladas H. Eléctrico** - Registro de coladas con carga eléctrica
- **Control de Temperaturas** - Monitoreo de temperaturas de vaciado

## Características
- Autenticación local (sin backend requerido)
- Dos niveles de usuarios: Supervisor y Hornero
- Formularios dinámicos con adición de filas
- Diseño responsivo para desktop, tablet y móvil
- Almacenamiento en localStorage
- Interfaz intuitiva y fácil de usar
